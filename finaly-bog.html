مشکلات بحرانی پروژه

Login/Session Handling

  • ورود با پسورد درست جواب نمی‌ده (Login with correct password fails)
  • ورود اشتباه پیام درست نشون نمی‌ده (Wrong login does not show correct error message)
  • سشن ساخته نمی‌شه → داشبوردها لود نمی‌شن (Session not created → dashboards not loading)
  • خروج درست عمل می‌کنه، مشکل فقط در لاگین/سشن هست (Logout works, login/session handling is broken)

Forget Password Flow

  • ایمیل ریست ارسال نمی‌شه (Reset email not sent)
  • توکن‌ها امن/یکبار مصرف نیستن (Tokens are not secure/one-time)
  • پسورد بعد از ریست هش نمی‌شه (Password not hashed after reset)
  • Rate limiting برای جلوگیری از abuse نداری (No rate limiting to prevent abuse)

Dashboard & Roles

  • داشبوردها بعد از لاگین بالا نمیان (Dashboards do not load after login)
  • نقش‌ها و دسترسی‌ها اعمال نمی‌شن (Roles and permissions not applied)
  • محافظت از صفحات برای کاربر لاگین نشده کار نمی‌کنه (Page protection for non-logged users not working)
  • رابط کاربری (جدول، دکمه‌ها) درست نیست (UI elements like tables/buttons are broken)

Front-end (UI/UX)

  • HTML و CSS نامعتبر یا ناقص (Invalid/incomplete HTML and CSS)
  • CSS بین صفحات یکدست نیست (CSS is inconsistent across pages)
  • دکمه‌ها hover/focus ندارن (Buttons lack hover/focus states)
  • تصاویر alt ندارن (Images missing alt attributes)
  • فوتر/هدر درست نمایش داده نمی‌شن (Footer/Header not displayed correctly)
  • CSS minify نشده (CSS not minified)

Security & DevOps

  • مدیریت سشن امن نیست (Session handling is insecure)
  • HTTPS در محیط عملیاتی اجباری نیست (HTTPS not enforced in production)
  • بکاپ دیتابیس نداری (No database backup)
  • لاگ ارورها مانیتور نمی‌شه (Error logs not monitored)
  • پروژه روی Git نیست (Project not on Git)
